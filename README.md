## Run Locally

Clone the project

```bash
  git clone https://github.com/tiedsandi/crwn-tech
```

Go to the project directory

```bash
  cd crwn-tech
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
